# منصة إرتقاء الرقمية - دليل الرفع على GitHub والتشغيل على Vercel

مرحباً بك! هذا الدليل يوضح لك بالخطوات السهلة والواضحة كيفية رفع مشروعك بالكامل على **GitHub** وتفعيله للاستضافة المباشرة على **Vercel** ليعمل الموقع بشكل تلقائي ومستمر مع كل تحديث.

---

## 🚀 الجزء الأول: رفع المشروع على GitHub

يمكنك رفع المشروع عن طريق تصدير التطبيق كملف ZIP من قائمة الإعدادات في AI Studio، أو مباشرة باتباع الخطوات التالية:

### 1️⃣ إنشاء مستودع (Repository) جديد على GitHub
1. اذهب إلى موقع [GitHub](https://github.com) وسجّل دخولك.
2. اضغط على زر **New** لإنشاء مستودع جديد.
3. قم بتسمية المستودع (مثلاً: `irteqa-platform`).
4. اختر المستودع ليكون **Public** أو **Private** حسب رغبتك.
5. **مهم جداً:** لا تقم باختيار "Add a README file" أو "Add .gitignore" لتجنب حدوث أي تعارضات عند الرفع.
6. اضغط على **Create repository**.

### 2️⃣ رفع الملفات باستخدام الكوماند لاين (Terminal) أو Git Bash
افتح سطر الأوامر في مجلد مشروعك ونفّذ الأوامر التالية بالترتيب:

```bash
# تهيئة المستودع المحلي
git init

# إضافة جميع الملفات للمستند المحلي
git add .

# تسجيل أول حفظ للمشروع
git commit -m "إطلاق الاصدار الأول - منصة إرتقاء"

# تغيير اسم الفرع الرئيسي ليكون main
git branch -M main

# ربط مشروعك المحلي بالمستودع على جيتهاب (استبدل الرابط برابط مستودعك الخاص)
git remote add origin https://github.com/YOUR_USERNAME/irteqa-platform.git

# رفع المشروع إلى جيتهاب
git push -u origin main
```

## 🌐 الجزء الثاني: تشغيل واستضافة الموقع على Vercel

منصة **Vercel** هي أفضل وأسهل طريقة لاستضافة مشاريع React و Vite مجاناً وبأداء خارق.

### 1️⃣ ربط الحساب والبدء
1. اذهب إلى موقع [Vercel](https://vercel.com) وسجّل دخولك باستخدام حسابك في **GitHub**.
2. في لوحة معلومات Vercel، اضغط على زر **Add New...** ثم اختر **Project**.

### 2️⃣ استيراد المشروع (Import Project)
1. ستظهر لك قائمة بمستودعات GitHub الخاصة بك.
2. ابحث عن المستودع الذي رفعته للتو (مثلاً: `irteqa-platform`) واضغط على زر **Import** بجانبه.

### 3️⃣ ضبط إعدادات المشروع (Project Settings)
سيتعرف Vercel على المشروع تلقائياً كـ **Vite/React**، ولكن تأكد من تطابق الإعدادات التالية للحصول على أفضل أداء:
- **Framework Preset**: اختر `Vite` (يتم تحديده تلقائياً).
- **Build and Output Settings**:
  - Build Command: `npm run build`
  - Output Directory: `dist`
  - Install Command: `npm install`
- **Environment Variables** (إذا كان لديك أي متغيرات بيئة إضافية مثل مفاتيح الـ API): قم بإضافتها في هذا القسم.

### 4️⃣ إطلاق الموقع (Deploy)
1. اضغط على زر **Deploy**.
2. انتظر ثوانٍ معدودة حتى تكتمل عملية البناء.
3. **مبارك! 🎉** سيمنحك Vercel رابطاً مجانياً ومباشراً لموقعك (على سبيل المثال: `irteqa-platform.vercel.app`) لتبدأ بمشاركته مع عملائك.

## 🛠️ تفاصيل إعدادات التوافق (Vercel Configuration)

لقد قمنا بإضافة ملف `vercel.json` بجذر المشروع لضمان التوافق المطلق مع خوادم Vercel:

- **إعادة توجيه المسارات (Rewrites)**: تضمن عدم ظهور خطأ **404** عند قيام المستخدم بعمل فريش أو إعادة تحميل للموقع أثناء التنقل بداخل الصفحات، حيث يقوم بتوجيه كافة الطلبات بذكاء إلى `index.html`.
- **توافق البناء (Vite Compatibility)**: قمنا بتهيئة وضبط مكتبة الحركات `motion` وملفات ستايل Tailwind v4 لتعمل بكفاءة وسرعة فائقة على الـ CDN الخاص بـ Vercel.

إذا واجهتك أي استفسارات أو كنت بحاجة لمزيد من المساعدة في تجهيز النطاقات المخصصة (Custom Domains)، يسعدنا دائماً تقديم الدعم لك!
